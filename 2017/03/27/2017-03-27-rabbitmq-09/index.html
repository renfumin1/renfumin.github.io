1.队列使用过的属性:
x-max-priority: 设置队列中所推送的消息最大优先级数字
x-expires: 设置队列过期时间 (expiration:用来设置队列中某条消息的过期时间; 队列上设置消息过期时间和消息上设置消息过期时间,优先级以较小的时间为准 )
设置消息过期,也可以不用设置队列过期时间
